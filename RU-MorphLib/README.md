# RU MorphLib

RU MorphLib - библиотека поддержки русификаций для Starsector.

Мод добавляет расширенные русские `$rulm_*` токены для диалогов: род, падежи, окончания, суффиксы, прошедшее время, состояния и готовые частые формы.

Текущая версия: `0.5.4`  
Токенов: `412`  
Форм: `824` masc/fem

## Зачем это нужно

В английском диалоге часто хватает `he / she / him / her`. В русском языке этого мало: нужны род, падеж, окончание, суффикс и форма глагола.

Без такой системы переводчик часто вынужден писать обезличенно:

```text
Собеседник смотрит на вас.
Пленный офицер молчит.
```

С RU MorphLib можно писать живее:

```text
Пленни$rulm_nikNom поднял$rulm_past взгляд.
Поговорить $rulm_withHim.
Освободить пленни$rulm_nikAcc.
```

В игре получится:

```text
Пленник поднял взгляд.
Поговорить с ним.
Освободить пленника.
```

или:

```text
Пленница подняла взгляд.
Поговорить с ней.
Освободить пленницу.
```

## Как это работает

RU MorphLib регистрирует `RuleTokenReplacementGeneratorPlugin` и добавляет свои `$rulm_*` токены в штатную систему token replacement Starsector.

Это значит, что для обычных `rules.csv`-диалогов не нужен отдельный JAR-bridge.

Обычно работает в:

```text
rules.csv text
rules.csv options
AddText
SetTooltip
AddOption / SetOption, если строка проходит через стандартную замену токенов
```

Может не сработать в полностью кастомном hardcoded UI, если мод выводит текст напрямую из JAR без стандартной подстановки токенов.

## Установка

1. Распаковать `RU MorphLib` в папку `Starsector/mods/`.
2. Включить мод в лаунчере Starsector.
3. Если переводимый мод использует `$rulm_*`, добавить RU MorphLib в зависимости этого мода.

## Как добавить зависимость

В `mod_info.json` переводимого мода добавить:

```json
{
  "id": "ru_morphlib",
  "name": "RU MorphLib"
}
```

Пример с уже существующими зависимостями:

```json
"dependencies": [
  {
    "id": "lw_lazylib",
    "name": "LazyLib"
  },
  {
    "id": "MagicLib",
    "name": "MagicLib"
  },
  {
    "id": "ru_morphlib",
    "name": "RU MorphLib"
  }
]
```

## Совместимость

RU MorphLib не заменяет vanilla-токены Starsector. Старые токены вроде `$heOrShe`, `$hisOrHer`, `$himOrHer`, `$playerName` продолжают работать как раньше.

RU MorphLib добавляет новый namespace:

```text
$rulm_*
```

Это позволяет использовать старые и новые токены в одной строке.

## Важно

RU MorphLib не переводит моды автоматически. Это инструмент для переводчиков. Качество растёт там, где переводчик осознанно использует `$rulm_*` в подходящих местах.
