# Unknown Skies - русификация

Неофициальная русификация мода `Unknown Skies` для `Starsector`.

Перевод находится в тестировании и не считается окончательным. Возможны неточные формулировки, редкие английские остатки и будущие правки по скриншотам из игры.

## Для какой версии

- `Starsector`: `0.98a-RC8`
- `Unknown Skies`: `3.0.3`
- Зависимости оригинального мода: `LunaLib`, `MagicLib`

## Что переводит архив

Русификация затрагивает только player-visible текст мода:

- подсказки загрузки;
- строки UI и tooltip-эффектов;
- описания планет;
- названия и описания планетарных условий;
- названия типов планет;
- настройки `LunaLib` для `Unknown Skies`;
- описание мода в списке модов.

Архив не содержит оригинальный мод целиком и не заменяет его.

## Установка

1. Установите оригинальный `Unknown Skies 3.0.3`.
2. Убедитесь, что установлены зависимости `LunaLib` и `MagicLib`.
3. Распакуйте архив русификации поверх папки мода `Unknown Skies`.
4. Подтвердите замену файлов.
5. Запустите игру.

## Важно

Это localization-only архив. Внутри находятся только изменённые файлы перевода.

Не устанавливайте русификацию как отдельный мод в папку `mods`. Её нужно накладывать поверх оригинального `Unknown Skies`.

## Статус перевода

Текущая версия - тестовая. Перевод будет дорабатываться по найденным ошибкам, скриншотам и замечаниям игроков.

Если нашли английский остаток, странную формулировку или проблему с отображением, лучше приложить скриншот и указать, где именно это найдено.
