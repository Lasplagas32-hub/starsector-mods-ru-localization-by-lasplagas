# Take No Prisoners - русификация

Фанатская русификация мода `Take No Prisoners` для `Starsector`.

`Take No Prisoners` добавляет возможность брать в плен вражеских офицеров после боя, просматривать список пленников через способность `View Prisoners` и вести с ними отдельные диалоги. У разных фракций есть свои варианты разговоров, реакции и исходы. Пленника можно освободить, казнить, потребовать выкуп, попытаться нанять или использовать особый фракционный вариант.

## Версия

- Версия мода: `Take No Prisoners 0.17.1`
- Тип архива: `localization-only`
- Установка: поверх оригинального мода

Архив русификации не содержит полный оригинальный мод. Внутри находятся только измененные файлы перевода.

## Ссылки

- Оригинальный устаревший GitHub-репозиторий: https://github.com/wispborne/takenoprisoners
- Обновленная версия мода: https://fractalsoftworks.com/forum/index.php?topic=30039.0

## Установка

1. Установите оригинальный `Take No Prisoners`.
2. Распакуйте архив русификации поверх папки мода.
3. Подтвердите замену файлов.
4. Запустите игру и проверьте, что мод включен в лаунчере.

## Требования

Требуются те же зависимости, что и у оригинального мода:

- `MagicLib`
- `LazyLib`

## Что переведено

- интерфейс пленников;
- способность `View Prisoners`;
- кнопки действий с пленниками;
- основные диалоги пленников;
- фракционные ветки:
  - `Luddic Path`;
  - `Pirates`;
  - `Hegemony`;
  - `Tri-Tachyon`;
  - `Sindrian Diktat`;
  - `Persean League`;
- сообщения, подсказки и часть hardcoded-строк внутри `TakeNoPrisoners.jar`.
