# RU MorphLib - справочник токенов

Версия: `0.5.4`  
Токенов: `412`  
Форм: `824` masc/fem

## Как читать названия падежей

```text
Nom - именительный: кто?
Gen - родительный: кого?
Dat - дательный: кому?
Acc - винительный: кого?
AccAnim - винительный одушевлённый: пленного / пленную
AccInan - винительный неодушевлённый: пленный / пленную
Ins - творительный: кем?
Prep - предложный: о ком?
```

## Два слоя токенов

### Слой 1 - морфемы

Это окончания, суффиксные хвосты, времена и краткие формы. Их пишут слитно с основой:

```text
пленни$rulm_nikAcc -> пленника / пленницу
поднял$rulm_past -> поднял / подняла
пленн$rulm_adjAccAnim -> пленного / пленную
```

### Слой 2 - готовые формы

Это готовые слова или фразы:

```text
$rulm_withHim -> с ним / с ней
$rulm_prisonerAcc -> пленника / пленницу
$rulm_whichAccAnim -> которого / которую
```
