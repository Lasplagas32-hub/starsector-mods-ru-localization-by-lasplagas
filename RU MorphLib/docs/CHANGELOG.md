# RU MorphLib changelog для релиза

## 0.5.4

- Русское описание мода во вкладке модов Starsector.
- `412` токенов.
- `824` masc/fem форм.
- Подтверждена работа через `RuleTokenReplacementGeneratorPlugin`.
- Подтверждён демонстрационный UAF/Nia-диалог.
- Добавлены дубли vanilla gender-токенов в namespace `$rulm_*`.
- Добавлены расширенные морфемные модели, причастия, состояния, притяжательные формы и готовые частые формы.

## Политика совместимости

- Старые токены не удалять.
- Старые токены не переименовывать.
- Смысл старых токенов не менять без крайней необходимости.
- Новые версии должны добавлять возможности без поломки старых переводов.
