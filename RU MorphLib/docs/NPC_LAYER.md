# NPC Layer - `$rulm_*`

`$rulm_*` отвечает за текущего NPC, обычно `activePerson`.

Используйте его для пленника, офицера, контакта, барного собеседника или другого персонажа, которого текущий диалог передал как активного.

## Не путать с игроком

```text
$rulm_* - текущий NPC
$rulp_* - игрок
```

Если использовать `$rulm_*` для игрока, можно получить род и формы текущего собеседника вместо капитана.

## Частые токены

| Токен | Мужская форма | Женская форма | Когда использовать |
|---|---|---|---|
| `$rulm_he` | он | она | простое местоимение |
| `$rulm_He` | Он | Она | начало предложения |
| `$rulm_himAcc` | его | её | винительный: вижу кого |
| `$rulm_himDat` | ему | ей | дательный: дать кому |
| `$rulm_himGen` | его | её | родительный или притяжательная форма |
| `$rulm_himAfterPrepGen` | него | неё | если предлог уже написан: `для $rulm_himAfterPrepGen` |
| `$rulm_withHim` | с ним | с ней | готовая форма с предлогом |
| `$rulm_aboutHim` | о нём | о ней | готовая форма с предлогом |
| `$rulm_thisNom` | этот | эта | этот пленник / эта пленница |
| `$rulm_thisAccAnim` | этого | эту | вести этого / эту |
| `$rulm_prisonerNom` | пленник | пленница | нейтральное слово для TNP-like сцен |
| `$rulm_prisonerAcc` | пленника | пленницу | действие над пленником |
| `$rulm_speakerNom` | собеседник | собеседница | когда слово пленник не подходит |
| `$rulm_capturedState` | захвачен | захвачена | готовое состояние |
| `$rulm_hiredState` | нанят | нанята | готовое состояние |
| `$rulm_freedState` | освобождён | освобождена | готовое состояние |


## Н-формы после предлогов 0.6.14b

Обычные формы `$rulm_himGen` и `$rulm_himAcc` дают `его/её`. После многих русских предлогов нужна н-форма: `него/неё`, `нему/ней`, `ним/ней`, `нём/ней`.

| Токен | Мужская форма | Женская форма | Когда использовать |
|---|---|---|---|
| `$rulm_himAfterPrepGen` | него | неё | после предлогов с родительным: `для`, `без`, `от`, `у`, `ради` |
| `$rulm_himAfterPrepAcc` | него | неё | после предлогов с винительным: `в`, `на`, `за`, `через` |
| `$rulm_himAfterPrepDat` | нему | ней | после `к`, иногда `по` |
| `$rulm_himAfterPrepIns` | ним | ней | после `с`, `перед`, `над`, `за` |
| `$rulm_himAfterPrepPrep` | нём | ней | после `о`, `в`, `на`, `при` |

Хорошо:

```text
Никто сегодня не вступится за $rulm_himAfterPrepAcc.
Поговорить с $rulm_himAfterPrepIns.
```

Плохо:

```text
Никто сегодня не вступится за $rulm_himAcc.
Поговорить с $rulm_himIns.
```

В простых частых случаях можно брать готовые формы целиком: `$rulm_forHim`, `$rulm_withHim`, `$rulm_aboutHim`, `$rulm_toHim`.

## Хороший пример

```text
$rulm_GuyOrGirlNom едва заметно вздрагивает. Конвой ведёт $rulm_himAcc к шаттлу.
```

Возможный вывод:

```text
Парень едва заметно вздрагивает. Конвой ведёт его к шаттлу.
Девушка едва заметно вздрагивает. Конвой ведёт её к шаттлу.
```

## Осторожный пример

```text
$rulm_thisNom $rulm_contactNom ждёт ответа.
```

Мужская форма звучит хорошо: `этот контакт`. Женская может звучать плохо: `эта контакт`. В таком случае перестройте фразу:

```text
Собеседник ждёт ответа. Связаться $rulm_withHim?
```

RU MorphLib даёт формы, но финальную фразу всё равно редактирует переводчик.
