# Case Selection Guide

| Суффикс | Падеж | Когда нужен |
|---|---|---|
| `Nom` | именительный | кто? что? |
| `Gen` | родительный | кого? чего? нет кого? |
| `Dat` | дательный | кому? чему? |
| `Acc` | винительный | кого? что? |
| `AccAnim` | винительный одушевлённый | видеть пленника, этого офицера |
| `AccInan` | винительный неодушевлённый | видеть этот объект |
| `Ins` | творительный | кем? чем? |
| `Prep` | предложный | о ком? о чём? |


## Как выбирать

- Нужен подлежащее: `Nom`.
- Нужен объект действия: чаще `Acc` или `AccAnim`.
- Нужна фраза `дать кому`: `Dat`.
- Нужна фраза `нет кого`: `Gen`.
- Нужна фраза `с кем`: готовые формы вроде `$rulm_withHim` или `Ins`.
- Нужна фраза `о ком`: готовые формы вроде `$rulm_aboutHim` или `Prep`.


## Местоимения после предлогов

Для русских местоимений после предлога часто нужна н-форма.

| Нужно | Используйте | Пример |
|---|---|---|
| для него/неё, без него/неё, от него/неё | `$rulm_himAfterPrepGen` | `для $rulm_himAfterPrepGen` |
| за него/неё, через него/неё | `$rulm_himAfterPrepAcc` | `за $rulm_himAfterPrepAcc` |
| к нему/ней | `$rulm_himAfterPrepDat` | `к $rulm_himAfterPrepDat` |
| с ним/ней, перед ним/ней | `$rulm_himAfterPrepIns` | `с $rulm_himAfterPrepIns` |
| о нём/ней, в нём/ней | `$rulm_himAfterPrepPrep` | `о $rulm_himAfterPrepPrep` |

Если предлог не нужен отдельно, проще взять готовый токен: `$rulm_forHim`, `$rulm_withHim`, `$rulm_aboutHim`, `$rulm_toHim`.

## Пример

```text
$rulm_GuyOrGirlNom смотрит на вас.
Конвой ведёт $rulm_himAcc к шаттлу.
Поговорить $rulm_withHim.
```
