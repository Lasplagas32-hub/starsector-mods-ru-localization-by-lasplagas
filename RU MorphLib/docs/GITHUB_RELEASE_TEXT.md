# GitHub Release Text - RU MorphLib 0.6.14b

## RU MorphLib 0.6.14b

Релиз русской морфологической библиотеки для переводов Starsector.

### Что нового

- Добавлены отдельные н-формы местоимений после предлога: `$rulm_himAfterPrepGen`, `$rulm_himAfterPrepAcc`, `$rulm_himAfterPrepDat`, `$rulm_himAfterPrepIns`, `$rulm_himAfterPrepPrep`.
- Они закрывают случаи вроде `для него/неё`, `к нему/ней`, `с ним/ней`, `о нём/ней`, когда предлог уже находится в переведённой строке.
- Готовые формы с предлогами, например `$rulm_forHim`, `$rulm_withHim`, `$rulm_aboutHim`, остаются в таблице и подходят для простых случаев.
- Обновлена документация для переводчиков и полная таблица токенов.

### Что уже умеет библиотека

- `$rulm_*` - род и падежи текущего NPC, пленника, офицера или контакта.
- `$rulp_*` - обращение к игроку: `Сэр`, `Мэм`, `Капитан`.
- `$ruln_*` - формы слов после чисел, ресурсов игрока и memory-чисел.
- Dynamic suffix layer - окончания для реальных dynamic sources.
- `RuMorphQuantity` - Public Quantity API для Java/Kotlin/JAR/UI-кода.

### Важно

RU MorphLib не переводит строки автоматически. Это инструмент для переводчиков и моддеров, чтобы русские строки в Starsector нормально согласовывались по роду, падежу и числу.

Статичные числа переводятся руками. Неизвестные dynamic numbers из чужих модов требуют аудита конкретного мода.
