# Roadmap

## Stable now

- NPC morphology tokens.
- Player address tokens.
- Quantity forms for fixed, player-resource and memory-source numbers.
- Route-first dynamic suffix layer.
- Public Quantity API.
- Case tokens for `парень/девушка` and `мужчина/женщина`.

## Possible future work

- Number formatting helper for raw floats like `9375.0`.
- Resource/UI helper for icon + required amount + available amount.
- More ready-made person-word token groups, if real translation cases need them.
- More per-mod audit recipes.

## Not planned as automatic magic

- Full automatic translation.
- Guessing unknown numbers from arbitrary text.
- Replacing translator judgement in dialogue scenes.
